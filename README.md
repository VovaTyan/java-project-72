# Веб-сайт на базе фреймворка Javalin для SEO анализа сайтов. Помогает вести данные в SQL Postgres в форме таблицы

[Развёртывание в Railway](web-production-a198.up.railway.app)

### Сборка проекта:
```bash
make build
```

### Сборка исполняемого файла jar:
```bash
make install
```

### Запуск приложения:
```bash
make start
```

### Тест:
```bash
make test
```

### Чистка:
```bash
make clear
```
